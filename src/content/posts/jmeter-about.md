# 简介

Apache JMeter 是 Apache 组织开发的基于 Java 的压力测试工具。用于对软件做压力测试，它最初被设计用于 Web 应用测试，但后来扩展到其他测试领域。是一个应用广泛的性能测试工具。

[link](https://jmeter.apache.org/)
